# Dashboard IoT Frontend

Application Angular pour visualiser les données IoT en temps réel.

## Installation

```bash
npm install
```

## Démarrage

```bash
ng serve
```

Ouvrir http://localhost:4200

## Build

```bash
ng build
```

## Fonctionnalités

- Dashboard temps réel
- Affichage température et humidité
- Statistiques et historique
- WebSocket pour mises à jour automatiques
